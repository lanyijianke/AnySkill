# Hello World Skill

这是一个示例 skill，展示正确的文件格式。

## 格式说明

每个 Skill 文件必须包含 YAML frontmatter：
- `name`: Skill 名称（小写、连字符）
- `description`: 描述它做什么、什么时候用

## 你的第一步

1. 删除这个文件
2. 创建你自己的 `.md` 文件或文件夹
3. Push 到 GitHub，`index.json` 会自动生成
